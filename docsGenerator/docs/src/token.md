# The IO Token

## Overview

IO is the multifunction SmartWeave Token (SWT) that powers The AR.IO Network and its suite of permaweb applications. The IO Token (É¸) has many uses, including:

- protocol incentives,

- staking by gateways

- payments for services like the Arweave Name System (ArNS).

The token acts as a permissionless and censorship resistant medium of common value for the network.
